# Bank Performance Dashboard (Power BI)

## Project Overview

This project presents an interactive **Bank Performance Dashboard** developed using Power BI. The dashboard provides a comprehensive analysis of banking transactions, loan distribution, customer demographics, and financial trends.

## Objectives

The goal of this project is to analyze banking data and provide insights into:

* Transaction performance
* Loan distribution across regions
* Customer demographic analysis
* Monthly financial trends
* Currency usage in transactions

## Dashboard Features

### 1. KPI Indicators

* Total Transaction Amount
* Total Loan Amount
* Customer ID
* First Transaction ID

These KPIs provide a quick overview of key financial metrics.

### 2. Loan Distribution by State

A bar chart showing how loan amounts are distributed across different states, helping identify high-demand regions.

### 3. Transaction Distribution by Gender

A pie chart displaying financial activity between male and female customers.

### 4. Monthly Transaction Trends

A line chart that tracks transaction activity throughout the year to identify seasonal trends.

### 5. Loan Amount by Account Type

A column chart comparing loan distribution across different account types:

* Savings
* Checking
* Business
* Student

### 6. Currency Distribution

A donut chart showing transactions across multiple currencies such as PKR, USD, AED, SAR, CAD, GBP, and AUD.

### 7. Regional Transaction Performance

A horizontal bar chart displaying total transaction amounts by state.

## Tools Used

* Power BI
